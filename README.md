# library_management_using_MySql
The Library Management System is a web-based application developed using PHP and MySQL to automate and simplify library operations. The system provides role-based access for Admin and Student, ensuring efficient management of books, students, and library records.

Admin Module:
Secure admin authentication
Add, update, and delete book records
Add and remove student accounts
View complete student profiles and borrowing history

Generate reports including:
Overdue book reports
Student activity reports
Complete library reports
Monitor issued and returned books
Automatically detect overdue books
Send notifications/messages to students for overdue returns
Manage fines and due dates

Student Module:
Secure student login
View personal profile and issued books
Search and browse available books
Request and reserve books online
View due dates and fine details
Receive overdue notifications from admin

Technologies Used:
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL

Key Features:
Role-based access control
CRUD operations for books and students
Automated overdue tracking and reporting
Notification system for overdue books
User-friendly and efficient library management

This project showcases practical implementation of PHP-MySQL integration, database design, CRUD operations, and report generation, making it suitable for academic and real-world use.
